核心能力

少点手工目录操作，多点确定性

模组体检

扫描 SMAPI 清单、依赖、版本和禁用状态，把可更新、缺依赖、重复模组放到可操作的位置。

Nexus 下载

接入 Nexus API 与网页确认流程，下载后进入 LSMA 的安装计划，减少浏览器和资源管理器之间的切换。

存档保护

在启动和更新前备份存档，支持导入、导出和检查存档进度，避免改模组前缺少回退点。

本地指南

结合游戏数据、图标缓存和存档状态，查询社区中心、配方、矿井、NPC 与收集进度。

使用路径

从检查到启动保持同一条链路

01

连接游戏目录

选择 Stardew Valley 安装目录，LSMA 自动识别 SMAPI、原版启动器和本地内容。

02

扫描模组与存档

检查模组状态、读取存档摘要，并把日志诊断结果汇总到桌面应用内。

03

备份后启动

按默认策略启动 SMAPI 或原版游戏，必要时先生成可恢复的存档备份。
